#### Windows Troubleshooting
| **Problem**                                                         | **Cause & Solution**                                                                                                                                                                                                                                                        |
| ------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Command Prompt says `python is not recognized`                      | **Cause:** The student forgot to tick "Add python.exe to PATH".<br><br>  <br><br>**Fix:**<br><br>  <br><br>1. Find the Python installer file again and run it.<br><br>  <br><br>2. Choose **"Modify"**.<br><br>  <br><br>3. Tick **"Add Python to PATH"** → Next → Install. |
| VS Code says "Python interpreter not selected" or can’t find Python | **Fix:**<br><br>  <br><br>1. Press `Ctrl` + `Shift` + `P`.<br><br>  <br><br>2. Type "Python: Select Interpreter".<br><br>  <br><br>3. Choose the version that says `Python 3.x.x` (usually at the top).                                                                     |
| Warning "Running as Administrator" or antivirus blocks the install  | **Fix:**<br><br>  <br><br>Right-click the installer → **"Run as administrator"**.<br><br>  <br><br>Alternatively, temporarily turn off antivirus software during the installation process.                                                                                  |
| Nothing happens when double-clicking the installer                  | **Cause:** The installer file may be corrupted.<br><br>  <br><br>**Fix:** Download the file again.                                                                                                                                                                          |
| Two Python versions are installed (old + new) and it is confused    | **Fix:**<br><br>  <br><br>Uninstall the old Python version from **Settings** → **Apps** → search "Python" → uninstall everything except the newest version.                                                                                                                 |
| VS Code opens but looks weird / extensions won’t install            | **Fix:**<br><br>  <br><br>Close VS Code completely → right-click its icon → **"Run as administrator"** once, then attempt to install the extension again.                                                                                                                   |
| Cannot type in the VS Code terminal (black screen)                  | **Fix:**<br><br>  <br><br>Click the dropdown arrow at the top of the terminal panel and choose **"Command Prompt"** or **"PowerShell"** instead of Git Bash.                                                                                                                |
#### macOS Troubleshooting

|**Problem**|**Cause & Solution**|
|---|---|
|Terminal says `python3: command not found`|**Cause:** The installer didn’t finish properly.<br><br>  <br><br>**Fix:** Run the `.pkg` file again and ensure the password is typed correctly.|
|Mac says “Visual Studio Code.app can’t be opened because Apple cannot check it for malicious software”|**Fix:**<br><br>  <br><br>Right-click the app icon → choose **"Open"** → click **"Open"** again on the warning prompt (this is only required once).|
|VS Code won’t start at all (bounces once in Dock then disappears)|**Fix:**<br><br>  <br><br>Go to **Applications** → right-click VS Code → **Show Package Contents** → **Contents** → **MacOS** → double-click "Electron" (this forces the application to open).|
|Extensions won’t install / spinning forever|**Fix:**<br><br>  <br><br>Close VS Code → reopen it while holding `Cmd` + `Shift` (this disables all extensions temporarily) → then install the Python extension again.|
|When running code: “Permission denied” or “zsh: command not found python3”|**Fix:**<br><br>  <br><br>Close the Terminal completely → reopen it. macOS requires a fresh terminal session to recognize the new installation.|
|The old macOS Python 2.7 is active and confusing VS Code|**Fix:**<br><br>  <br><br>Students must always use `python3` (not just `python`).<br><br>  <br><br>In VS Code press `Cmd` + `Shift` + `P` → "Python: Select Interpreter" → choose the path `/usr/local/bin/python3` or `/opt/homebrew/bin/python3`.|
|VS Code says “The Python extension failed to load”|**Fix:**<br><br>  <br><br>Restart the Mac. This resolves the majority of extension loading errors on macOS.|
|Cannot drag VS Code to the Dock|**Fix:**<br><br>  <br><br>Open VS Code first from the **Applications** folder, then right-click the icon in the Dock → **Options** → **Keep in Dock**.|
